# Arduino Projects – Elegoo Kit + Paul McWhorter Series

👋 Welcome! This is a collection of hands-on microcontroller projects built using an Elegoo, guided by [Paul McWhorter’s Arduino tutorials](https://www.youtube.com/playlist?list=PLGs0VKk2DiYw-L-RibttcvK-WBZm8WLEP).

Each folder contains:
- Code (`.ino` files)
- Circuit diagrams
- Photos or videos
- Project summaries and reflections

## 📚 Topics Covered
- Digital and analog I/O
- Sensors and actuators
- LCD and serial communication
- Real-time system control
- Embedded C/C++ concepts

## 🗂️ Projects

| Project | Description |
|--------|-------------|
| [01_BlinkLED](./Project01_BlinkLED) | Blink LED using digital output |
| [02_ButtonControl](./Project02_ButtonControl) | Use a button to control an LED |
| ... | ... |

## 🔧 Kit Used
- Elegoo The Most Complete Starter Kit
- Arduino IDE

## 🎯 Goal
To build foundational embedded systems skills in preparation for undergraduate research in spacecraft systems and microcontrollers.
